## What is This?

A full-stack interactive **building optimization and planning platform** covering all 9 Austrian federal states with 29 functional tabs. Built by ORION for Austrian builders, architects, and homeowners.

**Live Demo**: [orion-primordia.replit.app/world/architekt-at](https://orion-primordia.replit.app/world/architekt-at)

---

## Features (29 Tabs)

### Engineering Tools (Tabs 1-15)
| Tab | Tool | Description |
|:----|:-----|:------------|
| 1 | OIB-Richtlinien Engine | All 6 OIB guidelines with state-specific deviations |
| 2 | U-Wert Rechner | Thermal transmittance calculator |
| 3 | Kostenschätzung | Construction cost estimator by region |
| 4 | Förderungsfinder | Subsidy finder for all 9 states |
| 5 | Energieabschätzung | Energy estimation and optimization |
| 6 | Zeitplan | Construction timeline planner |
| 7 | Konkurrenzvergleich | Competitor comparison tool |
| 8 | Statik | Structural calculations (Eurocode) |
| 9 | Bautabellen | Building reference tables |
| 10 | KI-Bauberater | AI-powered building advisor |
| 11 | Taupunktrechner | Dew point calculator |
| 12 | Schallschutz-Rechner | Sound insulation calculator |
| 13 | Brandschutz-Checker | Fire protection checker |
| 14 | Baubook Materialien | 80+ materials with ecological data |
| 15 | Energieausweis | Energy certificate reference (A++ to G) |

### Reference & Subsidies (Tabs 16-29)
| Tab | Content |
|:----|:--------|
| 16-24 | Housing subsidies for all 9 federal states |
| 25 | Building code comparison across states |
| 26 | ÖNORM standards reference |
| 27-29 | Additional tools and references |
