## ISO 27001 (Information Security Management)

### Why It Matters
Enterprise customers (50+ people) **require** ISO 27001 for security requirements. Without it, we're excluded from many RFPs.

### What It Is
ISO 27001 is an international standard for information security management systems. It requires:
- Risk assessments
- Security policies
- Access controls
- Incident response procedures
- Regular audits

### Our Plan

#### Phase 1: Assessment & Gap Analysis (Current)
- [ ] Self-assessment using ISO 27001 framework
- [ ] Identify gaps vs standard
- [ ] Develop action plan
- Effort: 2 weeks (May-June 2026)

#### Phase 2: Policy & Process Development
- [ ] Write information security policy
- [ ] Implement access controls (MFA, SSO)
- [ ] Document incident response procedures
- [ ] Create backup/disaster recovery procedures
- Effort: 6-8 weeks (June-July 2026)

#### Phase 3: Implementation
- [ ] Roll out SSO (OAuth2/SAML)
- [ ] Implement audit logging
- [ ] Deploy DLP (Data Loss Prevention)
- [ ] Establish security monitoring
- Effort: 4-6 weeks (July-August 2026)

#### Phase 4: Certification
- [ ] Select external auditor
- [ ] Pre-audit (dry run)
- [ ] Final audit
- [ ] Certification
- Effort: 4 weeks (September-October 2026)

### Investment Required
- Consultant/Auditor: EUR 8,000-12,000
- Personnel time: ~200 hours
- Infrastructure/tools: EUR 2,000-3,000
- **Total: EUR 10,000-15,000**

### Business Impact
- ✅ Enterprise customers can now trust us with data
- ✅ RFP compliance for large deals
- ✅ EUR 50-100k deals suddenly feasible
- ✅ Competitive advantage vs non-certified competitors

### Target: Q4 2026

## SOC 2 Type II (Enterprise Security Standard)

### Why It Matters
Large US/international customers require SOC 2 Type II. It's the global standard for SaaS security.

### What It Is
SOC 2 audits 6 trust principles:
1. **Security:** Protect data from unauthorized access
2. **Availability:** System is available when needed
3. **Processing Integrity:** Data is complete and accurate
4. **Confidentiality:** Only authorized people access data
5. **Privacy:** Personal data is handled correctly
6. **Compliance:** Meets relevant laws/regulations

### Our Plan

#### Phase 1: Preparation (Q3 2026)
- [ ] Log all security controls
- [ ] Document procedures
- [ ] Set up monitoring
- Cost: EUR 5,000

#### Phase 2: Audit Period (Q3-Q4 2026, 6 months)
- [ ] External auditor validates controls
- [ ] Ongoing monitoring
- [ ] No interruptions to operations

#### Phase 3: Report & Certification (Q1 2027)
- [ ] SOC 2 Type II report issued
- Cost: EUR 15,000-20,000

### Impact
- Opens US market (99% of US enterprises require it)
- Competitive advantage
- EUR 200k+ deals become possible

### Target: Q1 2027
